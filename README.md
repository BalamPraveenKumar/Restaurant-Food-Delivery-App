# 🍽️ Restaurant Food Delivery App

A full-stack Food Ordering Platform built using the MERN stack.  
This project includes a customer application for browsing food and placing orders, along with an Admin Dashboard for managing inventory and tracking orders in real-time.

## 🚀 Project Overview

This application provides:

- A user-friendly interface for customers to browse food items
- Cart management and secure Stripe payment integration
- Order history tracking
- An Admin panel to manage food items and update order status
- Secure authentication using JWT

Built with scalability, performance, and real-world architecture in mind.

---

## 🏗️ Tech Stack

### 🔹 Frontend (Client & Admin)
- React.js (v19)
- Vite
- Context API (State Management)
- React Router DOM
- Axios
- CSS / CSS Modules

### 🔹 Backend (API)
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Bcrypt (Password Hashing)
- Stripe Payment Gateway Integration
- Multer (Image Upload Handling)

## 🔑 Key Features

### 👤 User Application

### 👨‍🍳 Admin Dashboard
- Add new food items
